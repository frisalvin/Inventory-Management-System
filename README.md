# 📦 Inventory Management System

Web-based Inventory Management System built to manage product stock, track incoming and outgoing goods, and generate reports efficiently.

---

## 🚀 Features

- 📋 Product Management (CRUD)
- 📦 Stock Tracking (In & Out)
- 📊 Stock Opname Management
- 📥 Import Data via CSV/Excel
- 📤 Export Reports to Excel
- 🧾 Export Reports to PDF
- 🔐 Authentication System (Login)

---

## 🛠 Tech Stack

**Backend**
- PHP (Native)
- MySQL

**Frontend**
- HTML5
- CSS3
- Bootstrap

**Tools**
- Apache (XAMPP)
- Excel / CSV Integration
